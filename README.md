## 🎯 Concept

> *Most law firm websites look generic — stock photos, default templates, and 
> uninspired blue-and-white palettes.*

**Vittorio & Associados** was built to break that pattern entirely: dark editorial 
design, restrained luxury aesthetics, and deliberate, confident animations that 
communicate authority without saying a word.

The goal: make someone think *"this firm charges R$50k+ per case"* — before reading 
a single line of copy.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 **Cinematic Loading Screen** | Branded intro with SVG line-draw animation and smooth curtain reveal |
| 🔲 **Glassmorphic UI** | Translucent cards, backdrop blur, subtle gold borders across practice areas & forms |
| 📜 **Scroll-Triggered Animations** | Staggered reveals, animated counters, parallax elements via intersection observer |
| ✒️ **Editorial Typography** | Bold serif headlines + clean sans-serif body — high contrast, premium feel |
| 📊 **Animated Results Section** | Count-up statistics on deals closed, success rate, years of practice |
| 📱 **Fully Responsive** | Mobile-first build with a flagship-quality desktop experience |

---

## 🎨 Design Direction
**Philosophy:** *restraint signals luxury.* Every design decision here removes 
noise rather than adds it.
